Lipid Parent Path

At the very top of the browser on the right hand side is the lipid parent path panel, it shows a breadcrumb of all the parent lipids leading directly to the currently select lipid record. For example having select the lipid PC 18:0/18:1 the lipid parent path looks as follows:

The elements of the path are click-able links to the parent lipids, clicking these will load the new lipid information and children information.
